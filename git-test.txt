Git tutorial:
Cài Git vào foler:
- Chuột phải vào folder chọn Git Bash here.
- Sau khi hiện ra termianl, thì gõ lệnh: git init
Đưa file vào Staging Area để Git theo dõi các thay đổi của file:
- git add <tên_file>
- git add .
Cam kết code trước khi gửi: git commit -m "Message" (Message tuỳ chọn)

Nếu đang ở bước tạo git cho foler:
-  git remote add origin https://github.com/khoa03052000/git-trainning-gcd.git

Đã remote xong, đẩy code lên trên internet: git push -u origin main


Đăng ký thông tin máy:
